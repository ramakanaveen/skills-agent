# Scripture and Religious Text Handler

## When to use
- User asks for specific verses, slokas, or chapters from religious texts
- Requests for Bhagavad Gita content (Sanskrit, English, Telugu, Hindi, etc.)
- Questions about religious scripture content
- Need to create study materials from religious texts

## Important Note
This agent does not have direct access to religious text databases. For specific
verses or slokas, the agent should:

1. Acknowledge the limitation honestly
2. Explain that it doesn't have access to the specific text requested
3. Suggest alternatives like:
   - Creating a template for the user to fill in
   - Providing guidance on where to find authentic sources
   - Offering to help format or organize scripture content if provided

## Procedure for Scripture Requests

1. **Acknowledge the request clearly**
   - Identify the specific text, chapter, and verse requested
   - Note the language preference if specified

2. **Be honest about limitations**
   - Explain that you don't have access to religious text databases
   - Avoid making up or approximating sacred content

3. **Offer helpful alternatives**:
   - Suggest authentic sources for the text
   - Offer to create study templates
   - Provide formatting help if they provide the content
   - Create organizational structures for study materials

4. **If content is provided**:
   - Help format it properly
   - Create study guides or analysis documents
   - Organize content into readable documents

## Rules
- NEVER fabricate or approximate religious text content
- Always be respectful when discussing religious materials
- Direct users to authentic sources for original texts
- Be honest about the agent's limitations
- Offer constructive alternatives

## Output
- Honest acknowledgment of limitations
- Suggested authentic sources
- Template documents for user input if requested
- Formatted religious study materials if content is provided
